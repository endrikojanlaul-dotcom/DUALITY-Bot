DUALITY (D·Y) Clan Management Bot

Discord bot for clan management focused on Gun Armory Tycoon (Roblox). Implements DE-P system, ranks, prestige, leaderboards, achievements, and utility features (bug reports, suggestions).

Features
- DE-P (experience) system with kill/death rules and rank multipliers
- Auto-calculated Rank System
- Prestige system with lifetime stats
- Leaderboards (DE-P, roster, prestige)
- Achievement system with role grants and announcements
- Utility systems: bug reports, suggestions, admin commands

Policies
- `docs/TERMS_OF_SERVICE.md` — Terms of Service for the Bot
- `docs/PRIVACY_POLICY.md` — Privacy Policy describing data collection and retention

- Notes
- Uses SQLite by default.
- Commands are implemented as hybrid commands (slash + prefix `!`).
